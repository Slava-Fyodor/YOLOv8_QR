# Интеллектуальная система распознавания QR-кодов на основе модели YOLOv8 с графическим интерфейсом

## Описание проекта

Данный проект представляет собой разработку интеллектуальной системы для распознавания QR-кодов, использующую передовую модель объектного детектирования YOLOv8. Система оснащена удобным графическим пользовательским интерфейсом (GUI), разработанным с использованием фреймворка Qt5 (PyQt5), что позволяет эффективно обрабатывать изображения и видеопотоки для обнаружения и декодирования QR-кодов.

Целью работы является создание надежного и производительного инструмента для автоматизации процесса распознавания QR-кодов, который может быть применен в различных областях, включая логистику, инвентаризацию, здравоохранение и розничную торговлю.

## Ключевые особенности

*   **Высокоточная детекция:** Использование модели YOLOv8 для точного и быстрого обнаружения QR-кодов.
*   **Интуитивно понятный GUI:** Графический интерфейс для легкой загрузки изображений, обработки видеофайлов и потокового видео с камеры.
*   **Гибкие настройки:** Возможность настройки параметров распознавания в реальном времени.
*   **Отчетность:** Отображение результатов распознавания и ключевых метрик производительности.

## Используемые технологии

*   **Язык программирования:** Python
*   **Фреймворк для объектного детектирования:** Ultralytics YOLOv8
*   **Графический интерфейс:** PyQt5
*   **Библиотеки:** OpenCV, PyTorch (или TensorFlow, в зависимости от версии YOLOv8), NumPy и другие.
